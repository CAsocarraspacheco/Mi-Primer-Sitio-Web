Etiquetas h

En esta primera entrega he aprendido a diferenciar las etiquetas h que sirven para la jerarquia de los titulos y van desde la h1 hasta h6.

Etiqueta de Bloque header.

header que hace este elemento, nos sirve para darle organizacion a la pagina a nivel de codigo, no visual, nos indica donte estan los titulos del encabezado.

Etiqueta de Bloque main.

main nos indica a nivel de codigo, no visual, donde esta el contenido principal de la pagina.

Etiquetas para los parrafos

elemento que se utilizan para escribir los parrafos como la etiqueta p, que ocupan toda la linea hasta donde alcance el ancho de la pantalla. La etiqueta span la diferencia es que sirve para escribir parrafos que no ocupan el ancho de la pantalla.

las etiquetas strong, em

con strong se ha resaltado las palabras importantes dentro del parrafo

Ademas con em se pueden resaltar frases o testos importantes y apaecen escritos en cursiva.

Etiqueta blockquote

este elemento blockquote sirve para citar un texto de un autor

Etiqueta pre

Elemento de bloque pre sirve para mostrar textos 
             preformateados ejemplo:
        Hola
            Aqui estamos diseñando la pagina.
         Suerte

Etiqueta div

Esta etiqueta div nos sirve como un contenedor para darle o utilizan una configuracion diferente de las otras secciones.

Etiquetas ol, ul, li

Elemento ul y el ol para hacer listas, ademas deben estar acompañados de la etiqueta li las veces de elementos que tenga la lista; la etiqueta ol crea la lista con un orden logico si la lista lo requiere,por ejemplo los colores de la banderan son los siguientes

  1. amarillo
  2. azul
  3. rojo

Elemento ul crea lista sin un orden logico si la lista no lo requiere,por ejemplo las partes del cuerpo

  • Cabeeza
  • tronco
  • extremidades
